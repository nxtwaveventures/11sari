# Eleven Sari

A modern e-commerce platform for luxury handcrafted sarees with blockchain verification.

## Features

- AI-powered stylist recommendations
- Blockchain-verified authenticity
- Interactive 3D product views
- Modern, responsive UI with smooth animations
- Personalized shopping experience

## Tech Stack

- Next.js
- React
- Framer Motion
- Three.js
- Tailwind CSS
- Blockchain integration

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

## Deployment

The site is deployed to GitHub Pages automatically when changes are pushed to the main branch.

To deploy manually:

```bash
npm run gh-pages
```
