# 11Sari - Premium Indian Clothing

A modern e-commerce platform for traditional Indian clothing, showcasing premium sarees and cultural wear.

## Features

- Responsive, modern design
- Product showcase with detailed descriptions
- About section highlighting our story and commitment
- Smooth animations and transitions
- Contact information for inquiries

## Getting Started

### Prerequisites

- Node.js 14.x or higher
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/nxtwaveventures/11sari.git
   cd 11sari
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Building for Production

```bash
npm run build
# or
yarn build
```
